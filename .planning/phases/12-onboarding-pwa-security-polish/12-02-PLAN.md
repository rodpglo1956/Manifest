  <action>
    1. Create src/lib/security/csrf.ts:
       - generateCsrfToken(): create random token, store in httpOnly cookie (csrf_token), return token
       - validateCsrfToken(request): compare X-CSRF-Token header with cookie value
       - withCsrf(handler) wrapper for API route handlers that auto-validates on POST/PUT/DELETE/PATCH
       - Token uses crypto.randomUUID() for generation

    2. Create src/lib/security/rate-limit.ts:
       - In-memory sliding window rate limiter using Map<string, {count, resetAt}>
       - rateLimit(identifier, limit=100, windowMs=60000): returns {success, remaining, resetAt}
       - authRateLimit(identifier): calls rateLimit with limit=10, windowMs=60000
       - Periodic cleanup of expired entries (every 5 minutes)
       - Identifier should be IP address or user ID
       - Export withRateLimit(handler, options?) wrapper for API routes

    3. Create src/lib/security/sanitize.ts:
       - sanitizeText(input): strip HTML tags, trim whitespace, limit length
       - sanitizeObject(obj, fields): apply sanitizeText to specified string fields
       - Use regex-based HTML stripping (no DOMPurify needed since we never render raw HTML -- all content goes through React which auto-escapes)
       - Export for use in server actions

    4. Create src/lib/security/headers.ts:
       - securityHeaders object with:
         - Content-Security-Policy: default-src 'self'; script-src 'self' 'unsafe-inline' 'unsafe-eval' (needed for Next.js); style-src 'self' 'unsafe-inline'; img-src 'self' data: blob: https:; font-src 'self'; connect-src 'self' https://*.supabase.co wss://*.supabase.co https://api.mapbox.com https://api.stripe.com; frame-src https://js.stripe.com https://checkout.stripe.com
         - X-Content-Type-Options: nosniff
         - X-Frame-Options: DENY
         - X-XSS-Protection: 1; mode=block
         - Referrer-Policy: strict-origin-when-cross-origin
         - Permissions-Policy: camera=(self), microphone=(), geolocation=(self)

    5. Update src/middleware.ts:
       - Import securityHeaders from security/headers
       - After auth check, apply all security headers to the response using response.headers.set() in a loop
       - Apply to supabaseResponse before return
  </action>
