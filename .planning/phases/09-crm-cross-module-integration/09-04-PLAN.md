Client component reusing MapLibre pattern from dispatch-map.tsx.

- Import MapGL, NavigationControl from react-map-gl/maplibre
- Same mapStyle: "https://tiles.openfreemap.org/styles/positron"
- Same initial viewport: US centered (lat 39.8, lng -98.5, zoom 4)
- For each lane: use getCityCoords(origin_city, origin_state) and getCityCoords(destination_city, destination_state) to get coordinates
- Draw arcs between origin/destination as SVG overlay or using a custom Layer with GeoJSON LineString features
  - Use react-map-gl Source/Layer components with GeoJSON FeatureCollection
  - Each lane as a LineString feature with coordinates [origin_lng, origin_lat] -> [destination_lng, destination_lat]
  - Style: line-color based on status (green for active, gray for inactive, orange for seasonal), line-width 2, line-opacity 0.7
- Origin/destination markers as small dots (green for origin, red for destination)
- Hover popup showing: "Origin City -> Destination City | X runs | $X.XX/mi"
- Legend: Active (green), Inactive (gray), Seasonal (orange)
- Handle lanes without geocodable cities gracefully (skip, show count below map)
