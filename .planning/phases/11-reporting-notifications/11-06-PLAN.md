1. Create src/components/reports/report-pdf.tsx with 4 PDF templates using @react-pdf/renderer:
   - Follow invoice-pdf.tsx styling patterns (professional blue #1e3a5f header, Helvetica font, clean table layout)
   - All templates share a common header (company name, report title, date range, generation date)

   a. ProfitLossReport({ org, data, dateRange }):
      - Revenue section: total revenue, revenue/mile, revenue/load
      - Expenses section: fuel costs, maintenance costs, total expenses
      - Profit section: net profit, profit/mile, profit margin %
      - Summary table with monthly breakdown if quarter/year range

   b. FleetReport({ org, data, dateRange }):
      - Fleet summary: total vehicles, active, in-shop, utilization %
      - Cost breakdown table: vehicle, maintenance cost, fuel cost, total cost, cost/mile
      - MPG summary, total miles driven

   c. ComplianceReport({ org, data, dateRange }):
      - Health score gauge value
      - Overdue items list with due dates
      - Inspection summary: total, passed, failed, conditional
      - DQ file completeness summary per driver

   d. DriverReport({ org, data, dateRange }):
      - Driver scorecard table: loads, miles, revenue, on-time %, MPG, safety, compliance
      - Top performer highlight
      - Summary statistics

2. Create src/app/api/reports/generate/route.ts (POST):
   - Accepts JSON body: { report_type: 'pnl' | 'fleet' | 'compliance' | 'driver', start_date, end_date }
   - Validates auth via getAuthContext()
   - Fetches relevant data from analytics_snapshots, driver_performance, etc. based on type
   - Fetches org details for header
   - Uses React.createElement (not JSX, per Phase 4 convention) to create PDF component
   - Renders to buffer via renderToBuffer from @react-pdf/renderer
   - Stores in Supabase Storage: reports/{org_id}/{report_type}_{start}_{end}.pdf (upsert)
   - Creates signed URL (1 hour expiry)
   - Returns JSON: { url, filename, generated_at }

3. Create src/lib/reports/actions.ts:
   - `getReportHistory()`: lists files in reports/{org_id}/ from Supabase Storage, returns sorted by created_at desc with signed download URLs
   - `generateReport(type, startDate, endDate)`: server action wrapper that calls the API route
