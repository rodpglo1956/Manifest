  <name>Task 1: Analytics snapshot API route and data helpers</name>
  <files>src/app/api/analytics/snapshots/route.ts, src/lib/analytics/snapshot-helpers.ts, tests/analytics/snapshot.test.ts</files>
  <behavior>
    - GET /api/analytics/snapshots returns last 30 days of snapshot data
    - GET /api/analytics/snapshots?days=7 returns last 7 days
    - aggregateWeeklyVolume groups snapshots by ISO week, sums booked + delivered
    - calculateCurrentMonthOnTime computes month-to-date on-time percentage
  </behavior>
  <action>
    **src/app/api/analytics/snapshots/route.ts:**
    - GET handler: authenticate user, get org_id from profile
    - Query daily_snapshots for org_id where snapshot_date >= (today - days) days, order by snapshot_date asc
    - Accept ?days query param (default 30, max 90)
    - Return JSON { snapshots: DailySnapshot[] }

    **src/lib/analytics/snapshot-helpers.ts:**
    - Export aggregateWeeklyVolume(snapshots: DailySnapshot[]): { week: string; booked: number; delivered: number }[] -- groups by ISO week number, sums loads_booked and loads_delivered
    - Export calculateCurrentMonthOnTime(snapshots: DailySnapshot[]): number -- filters to current month, sums on_time_deliveries / total_deliveries * 100, returns 0 if no deliveries
    - Export formatChartDate(dateStr: string): string -- returns "Mar 15" format for x-axis labels
    - Export formatCurrency(value: number): string -- returns "$1,234" format for tooltips

    **tests/analytics/snapshot.test.ts:**
    - Test aggregateWeeklyVolume with snapshots spanning 3 weeks
    - Test aggregateWeeklyVolume with empty array
    - Test calculateCurrentMonthOnTime with mixed on-time data
    - Test calculateCurrentMonthOnTime returns 0 for no deliveries
    - Test formatChartDate produces expected format
  </action>
  <verify>
    <automated>npx vitest run tests/analytics/snapshot.test.ts --reporter=verbose</automated>
  </verify>
  <done>Analytics API returns paginated snapshot data, helper functions aggregate and format correctly</done>
